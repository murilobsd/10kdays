# 10kdays
Calculadora para identificar qual dos 10 mil dias da web uma data representa,
em angular2.

## Instalação

```sheel
    $ git clone https://github.com/murilobsd/10kdays
    $ cd 10kdays
    $ npm i
    $ npm start
```
